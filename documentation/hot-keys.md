## Hot-Keys

The following shortcuts work in any editor window to modify or add features:

* `CTRL+B` Bold
* `CTRL+I` italics
* `CTRL+U` Underline
* `CTRL+1` Heading 1
* `CTRL+2` Heading 2
* `CTRL+3` Heading 3
* `CTRL+4` Heading 4
* `CTRL+5` Heading 5
* `CTRL+6` Heading 6
* `CTR+SHIFT+.` Superscript
* `CTRL+SHIFT+,` Subscript
* `CTRL+ATL+B` Reference
* `CTRL+ALT+L` Left Align
* `CTRL+ALT+R` Right Align
* `CTRL+ALT+E` Center
* `CTRL+ALT+J` Justify
* `CTRL+ALT+I` Student Input
* `CTRL+ALT+O` Computed Output
* `CTRL+ALT+P` Plot
* `CTRL+ALT+F` Function
* `CTRL+ALT+T` Table
